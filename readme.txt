Live website stats for wordpress sites.

== Description ==

Complete visitor stats for your WordPress website, tracking user page visit and activity made easy by this plugin. get stats for top things like browsers, operating system, screen size, countries, cities, referrer domain, pages, user, link type and many more.



### WP Live Statistics by http://pickplugins.com



= Features =
**Dashboard for stats**
Awesome graphical representation for top site factors like Top OS, Top Screen Size, Top Browsers, Top Page Terms, Top Countries, Top Cities, Top Reffrers, Top Pages, Top User.

**Visitor online**
Track who visit your website now and you will able to see logged-in user and guest information like city, country, browser, duration, operating system and many more.

**Visitor list**
List of all visit by date some useful data you will know about visitors.

**Filter top factor**
Filter stats for Operating System, Browser, Screen size, Referer, City, Country, URL, Link Type and many more.

**Page view stats**
You can see the page view stats by date, week, month, year and custom date range. Also you can display page view chart to your single post page or any custom post types.


== Installation ==

1. Install as regular WordPress plugin.
2. Go your Plugins setting via WordPress Dashboard and activate it.

after activating plugin you will see menu "WP Live Stats"

take a test by visiting any page of your website.
